# widget
1. создайте блок в вашей верстке
    ```sh
    <div id="widget"></div>
    ```
1. подключите скрипт
    ```sh

    <script src="https://dikiycy.github.io/widget/widget.js" type="text/javascript"></script>
    ```
1. Инициализируете запуск
    ```sh

    <script type="text/javascript">widgetMessage.init();</script>
    ```
